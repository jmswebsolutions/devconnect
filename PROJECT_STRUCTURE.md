# Project Structure

## Overview

DevConnect is a platform where developers can create profiles, showcase projects, list skills and connect with other developers.

The project is divided into frontend, backend and documentation.

## Folder Structure

```text
devconnect/
│
├── frontend/
│   └── Frontend application
│
├── backend/
│   └── Backend application
│
├── docs/
│   └── Project documentation
│
├── README.md
│
└── PROJECT_STRUCTURE.md
```
